# 心理健康管理后台 — 第一次框架梳理

> 梳理日期：2026-06-26

---

## 一、项目概览

这是一个 **心理健康管理后台系统**（"心理AI助手"），技术栈：

| 层面 | 技术 |
|------|------|
| 框架 | Vue 3 (beta) + TypeScript |
| 构建 | Vite 8 |
| UI 库 | Element Plus 2.14 |
| 路由 | Vue Router 5 |
| 状态管理 | Pinia 3 |
| HTTP | Axios（二次封装） |
| CSS | SCSS（sass-embedded） |

## 三、路由设计

```
/                          → redirect → /back/dashboard
/auth (AuthLayout)
  /auth/login              → Login.vue
  /auth/register           → Register.vue
/back (Layout)
  /back/dashboard          → dashboard.vue
  /back/knowledge          → knowledge.vue
  /back/consolation        → consolation.vue
  /back/mood               → mood.vue
```

- **AuthLayout**：左右分栏（左：品牌标语 + Logo，右：`<router-view />` 渲染登录/注册表单）
- **Layout**：上-左-右经典后台布局（`Leftnav` + `Navheader` + `<router-view />`）

## 四、数据流全景

### 4.1 请求链路

```
组件调用 API 函数
    ↓
api/admin.js  (login / categoryOption / articleList / fileInfo)
    ↓
utils/request.js  (Axios 实例)
    ├── 请求拦截器：自动从 localStorage 取 token 注入 headers['token']
    ├── baseURL: '/api' → Vite proxy → http://159.75.169.224:1235
    └── 响应拦截器：
        ├── code === '200' → 脱壳返回 data.data
        └── code === '-1' → 清 token，跳转登录页
```

### 4.2 认证流程（Login.vue）

```
用户输入用户名+密码 → submitForm()
  → formEl.validate() 校验
  → login(formData) 调用 /user/login
  → 成功：localStorage 存 token + userInfo
    → userType === 2（管理员）→ router.push('/back/dashboard')
```

### 4.3 Dashboard 页核心数据流

```
onMounted()
  ├── categoryOption() 获取分类树
  │     → 构建 categories[] 给下拉框用
  │     → 构建 categoryMaps{} 给表格列显示分类名
  │     → 注入 formItems[1].options（搜索表单的分类下拉）
  └── handleSearch() 首次加载文章列表

用户操作：
  搜索 → TableSearch @search → handleSearch(formData)
    → articleList({...pagination, ...formData})
    → tableData 更新 → el-table 渲染

  新增 → handleAdd() → dialogVisible = true → AddDialogs 弹窗
    → 填写表单 → 上传封面（fileInfo → /file/upload）
    → 提交 → emit('submit', payload) → 等待接入新增接口

  分页 → handlePageChange(page) → handleSearch()
```

### 4.4 侧边栏折叠数据流

```
Navheader.vue 点击折叠按钮
  → useAdminStore().toggleAdmin()
  → admin.ts: isAdmin = !isAdmin
  → Leftnav.vue: computed isAdmin
    → el-menu :collapse="isAdmin"
    → brand 区域 v-show="!isAdmin"
```

### 4.5 AddDialogs（新增文章弹窗）内部数据流

```
父组件传入：
  v-model:modelValue  → 控制弹窗显隐（computed get/set 双向绑定）
  :categories         → 分类下拉选项

表单数据：
  form (reactive) {
    title, categoryId, summary, tagArry, tags, coverImage, content, id
  }
  → 用户填写 → buildSubmitPayload() 组装
  → emit('submit', payload) 通知父组件

图片上传：
  el-upload → beforeUpload(校验) → handleUpload(自定义上传)
    → fileInfo(file, {businessId}) → /file/upload
    → 返回路径 → resolveImageURL → imageURL(预览) + form.coverImage(提交)
```
